# MMRAG – Multimodal RAG System
Performs PDF charts-diagrams extraction and summarizing
This project implements a multimodal retrieval augmented generation system
capable of processing PDFs, images, and extracted visual information.

Features
- PDF ingestion
- Image captioning
- Vector database indexing
- Semantic search
- RAG-based querying

Tech Stack
- Python
- ChromaDB
- Vision models
- LLM integration
